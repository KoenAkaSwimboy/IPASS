Detailed Description

HX711.

HX711 returns the data from the chip. The gain, GAIN, tries, maxT, Data, DT, SCK, times, avg, tare, calWeight and scale components are private attributes. The appropiate constructers and functions are provided

◆ read()

unsigned long hx711::read ( )

Get the data from the chip.

First this function checks if the chip is ready for retrieval if so, for 24 times the data is shifted out. Each SCK pulse shifts out one bit, for more information see the datasheet. When the data is shifted out, the function calls nextConver() to make the chip ready for the next conversion.
