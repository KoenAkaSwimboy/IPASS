Detailed Description

Weightscale.

This is a weightscale that gives back the number of kilograms that is on the physical weightscale. This class is a subclass of the hx711 class. The button, calWeight, maxTries, offset, Scale, gain and times components are private attributes. The appropriate constructers and functions are provided

◆ calibrate()

void weightscale::calibrate ( )

Calibrate the weightscale.

This function calibrates the weightscale, first it sets the tare weight. Then it asks the user to put the calibrate weight on the weightscale and press the button. When button is pressed the function gets the offset. After this it calculates the scale by dividing the given calWeight with the offset. Finally it sets the scale and stops.

start the application
