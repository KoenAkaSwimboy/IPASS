Calibrate the weightscale. 

This function calibrates the weightscale, first it sets the tare weight. Then it asks the user to put the calibrate weight on the weightscale and press the button. When button is pressed the function gets the offset. After this it calculates the scale by dividing the given cal\+Weight with the offset.
